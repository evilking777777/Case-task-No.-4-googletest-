# Кейс-задача №4 — Unit-тесты на GoogleTest (Queue / Heap / Binary Search Tree)

## Задание
Реализовать набор unit-тестов с использованием **GoogleTest** для следующих классов (интерфейс обязателен):
- Очередь (queue): `push`, `pop`
- Куча (heap): `push`, `pop`
- Бинарное дерево поиска: `push`, `pop`, `search`

> По условию “процедуры реализовывать не нужно, только интерфейс”.
В данном проекте добавлена минимальная реализация, чтобы тесты компилировались и запускались.

## Структура проекта
├─ CMakeLists.txt
├─ include/
│ ├─ queue.hpp
│ ├─ heap.hpp
│ └─ bst.hpp
└─ tests/
├─ test_queue.cpp
├─ test_heap.cpp
└─ test_bst.cpp

## Требования
- CMake 3.16+
- Компилятор C++ (MSVC / GCC / Clang)
- Интернет для скачивания googletest при первой сборке (FetchContent)

## Что проверяют тесты
- Queue: FIFO поведение (первый вошёл — первый вышел)
- Heap: извлечение максимального элемента (max-heap)
- BST: вставка и поиск элементов
